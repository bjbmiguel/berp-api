# BERP Core

O módulo central da plataforma **BERP (Banguela ERP)**.

## 📖 Sobre o projeto

BERP Core é o coração da plataforma BERP. Ele gerencia autenticação, segurança, configurações globais e a comunicação entre os módulos, permitindo que o sistema seja modular, distribuído e pronto para atender desde pequenos comércios até redes de hotéis e academias.

A plataforma BERP foi projetada para ser flexível e adaptável a diferentes domínios de negócio, incluindo:
- **Point of Sale (POS)**
- **Restaurantes**
- **Hotéis**
- **Academias**
- E muito mais...

## 🚀 Tecnologias utilizadas

- **Java** 17+
- **Spring Boot** 4.0+
- **PostgreSQL** 17+
- **Keycloak** (para autenticação e autorização)
- **Maven** (gerenciamento de dependências)
